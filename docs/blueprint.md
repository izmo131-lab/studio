# **App Name**: Ivora Solutions

## Core Features:

- Service Overview: Showcase Ivora Logistics' services: cargo transport, integral logistics, warehousing, distribution and customized logistics solutions.
- Consulting Services: Detail the benefits of Ivora Logistics' consulting services to optimize supply chain management.
- Custom Solutions Form: Allow potential clients to describe their logistics needs via form to request personalized logistics solutions.
- Testimonials: Show client testimonials.
- Contact Form: Provide a contact form for inquiries and service requests.
- AI Logistics Suggestion Tool: AI tool where users can input logistical challenges and receive data-driven suggestions for optimizing processes based on the database

## Style Guidelines:

- Primary color: Vibrant pink (#F06292) to embody the corporate identity and energy.
- Background color: Light pink (#FCE4EC), a desaturated tone of the primary color, for a soft backdrop.
- Accent color: Deep black (#000000) for contrast and highlights, reinforcing the brand's corporate identity.
- Body and headline font: 'Inter' (sans-serif) to ensure readability.
- Simple, clear icons that represent logistics, transportation, and warehousing, using the primary pink and black for emphasis.
- A clean, organized layout that effectively highlights Ivora Logistics' key services and value propositions.
- Subtle animations on key elements, such as service highlights or transitions, to create engagement.
